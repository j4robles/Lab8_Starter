Service workers and graceful degredation are related because service workers are what faciliate graceful degredation. When the network fails, service workers ensure the user can still receive whatever is available from the cache, providing limited functionality in the absence of expected technology. Without service workers fetching from cache, a user without a working network would simply not be able to use the site at all which is a very ungraceful degredation.
